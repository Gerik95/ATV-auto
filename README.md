ATV Auto

ATV Auto is a pet project SPA built for an automotive service business.

The project focuses on service presentation, simple navigation, responsive layout, and communication with potential clients through a website feedback form.

## Overview

This project was created as a practical frontend project based on a real business niche.  
The main idea was to build a structured and user-friendly single-page website for an automotive service business, with clear presentation of services and a simple way for users to send inquiries.

## Features

- Single-page application structure
- Clear presentation of services
- Feedback form for client inquiries
- Responsive design
- Simple and practical navigation
- Business-oriented UI structure

## Tech Stack

- React
- JavaScript
- HTML5
- CSS3

## What I Practiced

- Building business-oriented frontend interfaces
- Creating responsive page layouts
- Structuring service content
- Implementing feedback form logic
- Improving usability and interface clarity

## Project Goal

The goal of this project was to improve practical frontend skills by creating a usable SPA for an automotive service business and focusing on responsive design, structure, and user interaction.
